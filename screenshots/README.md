## Tips for Best Screenshots:
- Use full browser window (1920x1080 recommended)
- Clear browser zoom (100%)
- Make sure all UI elements are visible
- For dark mode, show contrast clearly
- For modal, ensure it's centered and visible

## Windows Screenshot Tool:
- Press `Win + Shift + S` to open Snipping Tool
- Select area and save directly to this folder
